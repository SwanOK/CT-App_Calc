# STAR Engine v3 - Data-Driven Cooling Tower Rating API

A first-principles thermodynamic and aerodynamic rating engine for counterflow cooling towers. This project reverse-engineers commercial CTI (Cooling Technology Institute) sizing algorithms, utilizing Tchebycheff integration, dynamic evaporative mass shrinkage, and non-dimensional fan affinity laws to calculate exact tower capability, L/G ratios, and system pressure drops.

The project is split into a high-performance Python/FastAPI backend and a sleek, distraction-free Vanilla HTML/JS frontend.

---

##  Quick Start: How to Run

### Prerequisites
You must have **Python 3.8+** installed on your system. 

run this to clone the repo:
```bash
git clone https://github.com/SwanOK/CT-App_Calc
```


### 1. Start the Python Backend (FastAPI)

Open your terminal or command prompt, navigate to the project folder, and run the commands for your operating system:

**Mac / Linux**
```bash
# 1. Create a virtual environment (optional but recommended)
python3 -m venv venv
source venv/bin/activate

# 2. Install the required physics and server libraries
pip install fastapi uvicorn pydantic psychrolib

# 3. Start the API server
uvicorn api:app --reload
```

**Windows**
```powershell
# 1. Create a virtual environment (optional but recommended)
python -m venv venv
venv\Scripts\activate

# 2. Install the required physics and server libraries
pip install fastapi uvicorn pydantic psychrolib

# 3. Start the API server
uvicorn api:app --reload
```
*You should see a message saying `Application startup complete.` The server is now running locally on port 8000.*

### 2. Launch the Frontend UI
You do not need a web server for the frontend. 
Simply open your file explorer, locate **`index.html`**, and **double-click it** to open it in Chrome, Safari, or Edge. 

Enter your tower parameters and click **Execute Simulation**. The browser will securely communicate with your local Python backend and render the results instantly.

##  Core Engineering Features

Unlike basic nominal-tonnage calculators, this engine calculates cooling tower performance using strict physical laws:

* **CTI ATC-105 Capability Bisection:** Iteratively calculates the exact Adjusted L/G ratio where Available KaV/L intersects Required KaV/L.
* **Dynamic Evaporative Shrinkage:** Tchebycheff integral dynamically subtracts water mass (~0.1% per °F of cooling) across the four temperature nodes to perfectly map the driving force (dh).
* **Safe Fan Affinity Scaling:** Utilizes strictly bounded parabolic regression to extract stable A, B, C aerodynamic constants for industrial fans (4ft to 24ft diameter) without running into polynomial hallucination.
* **Reverse Psychrometric Solver:** Derives Dry Bulb temperature from Wet Bulb and Relative Humidity on the fly using a microsecond bisection algorithm.
* **Dynamic HWT Correction:** Automatically applies KaV/L penalty multipliers for extreme hot water temperatures (>100°F) where standard Merkel assumptions break down.
